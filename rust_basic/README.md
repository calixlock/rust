#### rust 설정 rustup CMD

```sh
# rustup Rust 업데이트
rustup update

rustup install stable
// 안전한 버젼 설치하기

# 특정 버젼 설치
rustup install 1.76.0


# 나이트 버젼 설치
rustup default nightly

  # stable > beta > nightly
  # stable이 가장 안정정인 버젼
  # nightly는 최신, 최적화 x

# rustc버젼 체크(러스트 버젼체크로 보면 됩니다.)
rustc --version

```
